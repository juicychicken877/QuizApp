# The Witcher Quiz
Text your knowledge about The Witcher universum in this interactive React Quiz App

<img src="src/assets/GeraltThumbsUp.png">

## Project Configuration

1. Navigate to the project directory.
2. Run `npm install` to install dependencies.
3. After installation, run `npm run dev`.
4. The app will be available at [localhost:5137](http://localhost:5137).

## The Quiz is avaliable at [The Witcher Quiz](https://thewitcherquiz.netlify.app)
